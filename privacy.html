Политика конфиденциальности

Общая информация

Сервис symbols, далее — «Сервис», уважает приватность и не собирает персональные данные пользователей. Вся обработка текста происходит исключительно в браузере, и сервис не имеет доступа к содержимому текстов.

Какие данные мы не собираем

  • Сервис не сохраняет введённые тексты на наших серверах
  • Сервис не анализирует содержимое текстов
  • Сервис не передаёт тексты третьим лицам
  • Сервис не использует тексты для обучения моделей ИИ или других целей

Локальное хранение данных

Сервис использует localStorage вашего браузера для следующих целей:

  • Сохранение истории текстов, доступно только вам и только на вашем устройстве
  • Сохранение пользовательских настроек, тема оформления, видимость панелей
  • Сохранение словаря автодополнения, для улучшения функции автодополнения

Все эти данные хранятся исключительно на устройстве и не передаются на сервер.

Аналитика

Сервис использует анонимную аналитику для улучшения работы сервиса. Собираются только общие данные о посещаемости и использовании функций, без привязки к конкретным пользователям или их текстам.

Как очистить данные

Вы можете в любой момент очистить все локально сохраненные данные следующими способами:

  • Использовать кнопку "Очистить историю" для удаления истории текстов
  • Использовать кнопку "Очистить словарь" для удаления пользовательского словаря автодополнения
  • Очистить данные сайта в настройках браузера

Изменения в политике конфиденциальности

Сервис может обновлять политику конфиденциальности время от времени. Все изменения будут опубликованы на этой странице.
